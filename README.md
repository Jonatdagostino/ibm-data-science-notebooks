## Contents

- Data Science Ecosystem overview
- Exploratory Data Analysis (EDA)
- Data visualization with Matplotlib
- Basic statistical analysis with Python

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Purpose

These notebooks demonstrate foundational data science skills including data cleaning, visualization, and basic statistical reasoning.
